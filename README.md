# Интернет-магазин (Online Store)

Полноценное веб-приложение интернет-магазина, состоящее из backend API на Express и frontend на React.

## Технологии

### Backend
- Node.js
- Express.js
- CORS
- nanoid (генерация ID)
- Swagger / OpenAPI (документация)

### Frontend
- React 18
- Axios (HTTP клиент)
- CSS (чистый CSS, без препроцессоров)

## Установка и запуск

### Backend

```bash
cd backend
npm install
npm run swagger
